Image-Processing Line Follower Robot

Summary

This project involved building an image-processing line-follower robot using a Raspberry Pi. The original goal was to create a robot that could capture top-down views, detect obstacles, calculate object distances, and navigate from a source to a destination using the shortest path. The final implementation focused on an image-processing line follower with an attempt to integrate shortest-path behavior.

The robot used a Raspberry Pi, Pi Camera, motor driver, left and right motors, 12V battery, and switch. Instead of traditional infrared sensors, it relied on the Pi Camera video stream to identify the path and determine movement direction.

The software stack included Raspbian Buster, Python, RPi.GPIO, Picamera, OpenCV, Numpy, and Time. The vision pipeline defined a region of interest, detected black path regions, and used OpenCV processing such as Gaussian blur, thresholding, erosion, dilation, contour detection, and image moments to calculate x-axis and y-axis centroids for motor control.

A key technical challenge was making the robot robust against shadows and grey zones by using edge-detection-oriented computer vision techniques rather than relying only on grayscale thresholding, while exploring how the robot could travel available paths to identify the shortest route.
