About Me

I am a software engineer focused on AI, machine learning, data engineering, and software quality assurance. My work sits at the intersection of research and production: building RAG-based LLM applications, YOLO-powered computer vision models, automated web scraping pipelines, and data analysis workflows that turn raw information into practical decisions.

Alongside my AI and data work, I have strong hands-on experience in test planning, automation, defect lifecycle management, and release quality. I have published research in Scientific Reports and Computers in Biology and Medicine, and I am especially interested in scalable AI systems, biomedical NLP, real-time object detection, and tools that make software teams move faster with confidence.

What I Do

Machine Learning

I build machine learning and deep learning solutions for practical problems, including YOLO-based object detection, image processing, transfer learning, and segmentation workflows. I also work on model training, evaluation, optimization, and deployment.

Data Analysis

I collect, clean, transform, and analyze data to uncover useful patterns and support evidence-based decisions. My work includes web scraping, feature engineering, SQL, Python analysis, visualization, and automated data pipelines.

Manual & Mobile Testing

I design and execute manual, mobile, and exploratory tests from requirement analysis through release validation. I focus on finding critical issues early, documenting defects clearly, and improving product stability before production.

Automation Testing

I create and maintain automation workflows that reduce repetitive testing effort and improve release confidence. My experience includes mabl, Selenium, API validation, pilot automation initiatives, and mentoring teams on quality practices.

API Testing

I validate APIs for functionality, reliability, data correctness, and integration behavior. I write test scenarios, inspect request and response flows, analyze failures, and help ensure backend systems behave consistently.

Web Design & Development

I build web tools and applications with Python frameworks such as Django, Flask, and FastAPI, along with HTML, CSS, and JavaScript. I care about clean implementation, performance, compatibility, and maintainable user-facing workflows.
